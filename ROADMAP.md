# Roadmap

## 0.1 foundation

- Stable input validation and output schema.
- Conventional and Firth logistic models.
- Aggregate-first exports with explicit row-level opt-in.
- Synthetic HDP example, tests, and continuous integration.

## 0.2 validation

- Bootstrap optimism correction and confidence intervals.
- Calibration plots with uncertainty.
- Configurable predictor scaling and clinically meaningful units.
- Additional synthetic edge cases for rare outcomes and separation.

## 0.3 missing data

- Multiple-imputation integration points.
- Missing-data mechanism documentation.
- Pooled model and performance summaries where methodologically appropriate.

## Longer term

- Independent public-data demonstrations.
- Decision-curve analysis after methodological review.
- Translated documentation and teaching materials.
- A stable extension API for domain-specific cohort templates.

Priorities will be adjusted from public issue feedback and real maintenance
needs. The roadmap is not a promise of clinical validation.
